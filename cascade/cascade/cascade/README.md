## Quick Start

### Prerequisites
- Node.js 18+
- Python 3.11+
- Docker & Docker Compose
- Venice API credentials

### Installation

```bash
# Clone the repository
git clone https://github.com/venice-ai/cascade.git

# Install dependencies
cd cascade
npm run install:all

# Set up environment variables
cp .env.example .env
# Edit .env with your Venice API credentials

# Start development servers
npm run dev
```
